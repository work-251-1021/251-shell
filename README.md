# Virtual Computer Shell

A terminal-based virtual shell written in Python.

## Features

- User sign-up and login system
- User information stored in a local `login.txt` file
- Terminal-style command interface
- Multiple built-in commands
- Modular project structure for future expansion
- Cross-platform support using Python

## Upcoming Features

- Online package installation system
- Improved file management commands
- Enhanced user authentication
- More built-in shell utilities

## Technologies

- Python 3
- os
- subprocess
- Rich (Terminal UI)
- File handling

## Project Goal

The goal of this project is to simulate a lightweight command-line operating system environment using Python. It is designed as a learning project to improve knowledge of Python, terminal applications, file systems, and software architecture.
